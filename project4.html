Project Overview

For our diploma thesis we are building a form intelligence engine that recognises scanned correspondence, classifies each template, and extracts the key data points auditors care about. Every prediction is cross-checked against the original OCR text so reviewers keep full trust in the automation loop. [Thesis Requirements Brief, 2024]

The solution is split into two services: a classifier that detects the document type and extracts fields, and an analysis layer that flags mismatches between the AI output and the raw text. This setup keeps training data clean and lets us harden the model as new edge cases appear. [Service Architecture Overview, 2024]
