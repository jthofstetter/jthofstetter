AIvour was a Shopify-based website where I built a frontend that was optimized for both mobile and desktop users. The main goal was to allow people to order AI-generated images by entering a prompt and configuring certain parameters like the number of steps or the desired quality.

Once the user submitted their order and paid, the details (including prompt and configuration) would be sent to my system automatically. From there, I would process the request, generate the AI-based image, and deliver it back to the customer.

Essentially, it was an online store for AI artwork. The Shopify integration handled the e-commerce side (payments, cart, etc.), while my custom backend script took care of the prompt processing and image generation workflow.

This project taught me a lot about setting up an efficient flow between Shopify and an AI generation system. I also focused on making the design appealing and easy to navigate, ensuring the site was fully responsive for all devices.
