Project Overview

Over a three-month build sprint our team created Elster, a research environment that listens to the DEX Screener API, evaluates brand-new meme coins, and automatically purchases the most promising ones with virtual funds. Once positions are opened we can define stop-loss parameters or exit them manually to simulate a fully fledged trading workflow.

The architecture keeps things modular so that the fake-money layer can later be replaced with real transactions. I focused on clean data ingestion, extendable analysis modules, and a smooth operator dashboard so we can safely test automation ideas before putting money on the line.

Key Highlights

  • Live coin intake with filtering logic that scores new launches against custom KPIs.
  • Scenario-based paper trading including stop-loss presets and sell triggers.
  • Insightful telemetry for each trade so we can evaluate strategies faster.
