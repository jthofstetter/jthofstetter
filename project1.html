Crypto Project - Elster

Team
4
Rating
9/10
Cost
$0

We had a total of three months to come up with a project, and we called it “Elster.” The main idea was to use the DEX Screener API to automatically fetch the newest (meme) coins. Our system analyzes them according to certain criteria, and if they meet our requirements, the platform “buys” them with fake money.

After that, we can manually set stop-loss parameters or choose to sell them again. Essentially, it’s a semi-automated trading environment for testing new meme coins without risking real capital. However, the workflow is designed to be easily extendable to real transactions if needed.

We also developed a frontend to visualize and manage everything, using C# and Blazor for the application. By leveraging C# for the backend logic, we could quickly integrate the DEX Screener API, parse incoming data, and run our analysis algorithms.

Our team faced challenges regarding API rate limits and data normalization, but in the end, we successfully built a prototype that demonstrates how automated trading (or at least scanning for promising meme coins) could work in real life.

This was an excellent learning experience in combining API integration, automation logic, and a user-friendly UI to manage trades and settings.
